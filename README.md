# Customer-Behavior-dashboard
👩🏻‍💻 Customer Behavior Data Analyst Portfolio Project

This project is an end-to-end Data Analytics Portfolio Project built using real customer shopping behavior data. It covers the complete workflow followed by professional Data Analysts—starting from data cleaning and transformation to SQL insights, dashboard creation, and final reporting.

This project demonstrates my skills in:
📊 Python (EDA)
🗃 SQL (Business Insights)
📈 Power BI (Dashboarding)
📄 Report Writing & Business Understanding

📌 Project Overview

The goal of this project is to analyze how customers shop, what influences their purchase decisions, and which customer segments contribute the most to revenue.

The project workflow includes:

✅ 1. Data Preparation & EDA (Python)

Performed extensive data cleaning, feature engineering, and exploratory analysis.
Major steps included:

Handling missing values

Converting data types

Creating age groups

Creating customer segments: New, Returning, Loyal

Visual EDA to understand purchase trends

✅ 2. Business Analysis Using SQL

Write multiple SQL queries to answer real business questions such as:

Which gender generates more revenue?

What are the top-rated products?

Do subscribers spend more than non-subscribers?

Which products have the highest discount usage?

Are repeat buyers more likely to subscribe?

✅ 3. Power BI Dashboard

Built a fully interactive dashboard with:

Revenue breakdown

Gender-wise & age-wise spending

Top categories & products

Discount analysis

Customer segmentation

Subscription impact on revenue

✅ 4. Final Project Report

Summarized insights, recommendations, and business strategy ideas derived from data.
